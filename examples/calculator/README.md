# Calculator Application

A functional calculator demonstrating grid layout and button callbacks.

## Quick Start

```bash
cd examples/calculator
python calculator.py
```

## Features

- Basic arithmetic operations (+, -, *, /)
- Clear and backspace functions
- Decimal point support
- Keyboard input support
- Error handling for division by zero

## Created With PygubuAI

```bash
pygubu-create calculator 'calculator with number pad, display, and operation buttons'
```

## PygubuAI Features Demonstrated

- Grid layout management
- Multiple button callbacks
- Entry widget as display
- Keyboard event binding
- State management for operations
