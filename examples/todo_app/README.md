# Todo List Application

A complete todo list app demonstrating PygubuAI's natural language creation and AI-assisted development.

## Quick Start

```bash
cd examples/todo_app
python todo_app.py
```

## Features

- Add tasks with Enter key or button
- Delete individual tasks
- Clear all tasks
- Task counter
- Scrollable list

## Created With PygubuAI

```bash
pygubu-create todo 'todo app with entry, add button, scrollable task list, and clear button'
```

## PygubuAI Features Demonstrated

- Natural language UI generation
- Listbox with scrollbar
- Entry validation
- Multiple button callbacks
- Dynamic widget updates
