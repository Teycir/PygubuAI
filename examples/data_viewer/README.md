# Data Viewer

A data table viewer demonstrating Treeview widget for displaying structured data.

## Quick Start

```bash
cd examples/data_viewer
python data_viewer.py
```

## Features

- Treeview widget with multiple columns
- Column headers and sorting
- Search functionality
- Load and refresh data
- Record counter
- Status bar

## Created With PygubuAI

```bash
pygubu-create data_viewer 'data table with columns for name, age, city, search bar, and toolbar'
```
