# Login Form

A professional login form demonstrating form validation and password handling.

## Quick Start

```bash
cd examples/login_form
python login_form.py
```

## Features

- Username and password fields
- Password masking
- Remember me checkbox
- Form validation
- Enter key to submit
- Visual feedback for login status

## Created With PygubuAI

```bash
pygubu-template login_form login
```

## PygubuAI Features Demonstrated

- Template-based creation
- Entry widgets with validation
- Password entry (show='*')
- Checkbutton widget
- Label updates for feedback
- Button state management
