# Settings Dialog

A tabbed settings interface demonstrating notebook widgets and various input types.

## Quick Start

```bash
cd examples/settings_dialog
python settings_dialog.py
```

## Features

- Tabbed interface with Notebook widget
- Checkbuttons for boolean settings
- Combobox for theme selection
- Scale widget for numeric values
- Real-time value updates

## Created With PygubuAI

```bash
pygubu-template settings_dialog settings
```

## PygubuAI Features Demonstrated

- Notebook (tabbed) widget

- Combobox with readonly state
- Scale widget with callback
