## Allowed Characters

Use only standard ASCII characters:
- Letters: A-Z, a-z
- Numbers: 0-9
- Standard punctuation: . , ! ? : ; ' " - ( ) [ ] { }
- Standard symbols: @ # $ % & * + = / \ | _ ~
- Whitespace: spaces, tabs, newlines

## Enforcement

This rule applies to:
- All chat responses
- Code comments
- Documentation
- File content
- Error messages
- Status updates

Use clear, professional language without decorative characters.
